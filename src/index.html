Навыки

Что я использую в работе

HTML5

Верстаю с помощью методологии БЭМ, так же не упускаю возможность правильно сверстать сайт с точки зрения SEO по стандарту HTML5 от whatwg. Использую технологии Grid, Flexbox верстки в своих проектах. Так же имею небольшой опыт и понимание в верстке e-mail писем с помощью inline стилей и таблиц.

CSS3

При написании стилей использую препроцессоры такие как SCSS либо SASS. Создаю адаптивную, отзывчивую верстку под любые устройства. Практикую создание интересных анимации с помощью @keyframe. Собираю проекты по блоково, разделяя важные и независимые блоки на отдельные файлы.

Java Script

Имею базовый опыт в JS, чаще всего создаю не сложный код с помощью ChatGPT путем прописывания четкого алгоритма действий. Умею таким образом собирать самые популярные и не сложные вещи на сайте. В данный момент прохожу курс по JS и углубляюсь в эту сферу.

Jquery

Библиотеку Jquery знаю как ответвление от JS для каких либо анимаций на сайте. Удобно использую готовые решения с github, так же как и в JS. Имею базовые знания но не более.

Gulp

Сборщиком проектов Gulp пользуюсь постоянно, имею пару готовых темплейтов и работаю с ними исходя из потребностей в проекте. Понимаю как использовать npm пакеты, и подключать новые задачи.

GitHub

На Github стараюсь пушить все свои проекты которые сделаны не на CMS, так же загружаю свои лэндинги на GitHub Pages, здесь же и лежит это портфолио.

WordPress

Имею опыт в WordPress, создавал пару проектов на таких темах как WoodMart / Breakdance. Так же создавал E-commerce сайты через WooCommerce. В данный момент практикую установку собственных лендингов на WordPress, путем связки html и php. Имею небольшое представление о базе данных MySQL
